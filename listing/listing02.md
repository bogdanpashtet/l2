Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main
import (
	"fmt"
)

func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}
func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}
func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Функция defer вызывается перед завершением функции.

anotherTest():
При вызове функции anotherTest() значение "x" определено перед defer. 
Когда выполнение программы доходит до defer, то defer будет работать с текущим значением "х",
следовательно, будет выведено значение 1: при инициализации "х" получает значение 0, 
после дефера - 1, затем происходит return из функции.

test():
В этом случае переменная "х" является по сути глобальной, и при присваивании ей значении 
единицы - мы присваиваем ей значение единицы, а затем при вызове дефера инкрементируем 
это значение, тогда при возврате из функции инкрементируем значение глобальной переменной
и получаем 2.
```
