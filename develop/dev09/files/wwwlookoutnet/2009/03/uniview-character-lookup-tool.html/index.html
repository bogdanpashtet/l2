Uniview character lookup tool

12 Mar 2009
Richard Ishida has an online character lookup tool which is very nice. It’s called Uniview and it’s comparable to Babelmap in some functionality but it’s available online if that’s useful to you.

If you’re looking to use any of the characters I talk about for testing, you can find them through either one of these tools. Just type in the code point and then copy/paste the character.
